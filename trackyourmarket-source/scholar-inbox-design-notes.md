# Scholar Inbox UI Design Notes

## Key Design Patterns

### Color Scheme
- **Primary**: Red/Coral (#E53935 or similar)
- **Background**: Clean white (#FFFFFF)
- **Text**: Dark gray/black for readability
- **Cards**: White with subtle shadows
- **Accents**: Light gray borders

### Typography
- Large, bold headings with serif or elegant sans-serif
- Clean, readable body text
- Generous line spacing

### Layout
- Clean, centered content
- Card-based layout for content items
- Generous whitespace
- Simple navigation at top
- Feature cards in grid layout

### Paper/News Cards
- White background
- Title in bold
- Metadata (date, source) in smaller gray text
- Action buttons (like, bookmark) on right side
- Hover effects for interactivity
- "Likes" and "Reads" counts displayed

### Navigation
- Simple horizontal nav bar
- Tabs: Trending, My Papers, Starred, Semantic Search, Conferences
- Search on right side
- Clean, minimal design

### Key UI Elements
- Outlined icons (not filled)
- Subtle shadows on cards
- Rounded corners
- Clear visual hierarchy
- Date range selector/pagination
